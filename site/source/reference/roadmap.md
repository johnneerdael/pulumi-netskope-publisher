# Roadmap

Supported provider components:

- AWS
- Azure
- GCP
- Kubernetes
- vSphere
- ESXi Native
- Hcloud
- Nutanix
- OpenStack
- OVH Public Cloud
- Scaleway
- OCI
- Alicloud
- Proxmox VE
- DigitalOcean
- Vultr
- Exoscale
- UpCloud
- Stackit
- Equinix Metal
- Outscale
- OpenTelekomCloud
- TencentCloud
- Yandex Cloud

Planned follow-up:

- Build a fuller provider capability framework with provider metadata,
  stricter validation, and docs/example generation hooks.
- Enable full Hyper-V once `@pulumi/hyperv` is available through a stable
  package source.
- Submit the package to the public Pulumi Registry after the first tagged
  release uploads plugin archives.
