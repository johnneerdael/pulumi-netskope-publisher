# Roadmap

Supported provider components:

- AWS
- Azure
- GCP
- Kubernetes
- vSphere
- ESXi Native
- Hcloud
- Nutanix
- OpenStack
- OVH Public Cloud
- Scaleway
- OCI
- Alicloud
- Proxmox VE

Planned follow-up:

- Enable full Hyper-V once `@pulumi/hyperv` is available through a stable
  package source.
- Submit the package to the public Pulumi Registry after the first tagged
  release uploads plugin archives.
